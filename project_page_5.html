About

The News homepage is a self-study project built with HTML, CSS, SCSS, and JavaScript. This News homepage project was built within the Riga TechGirls program "Tech Mentor" 2023 and with the help and advice of my mentor. It is a static page - only the main page without active links. This project was a great opportunity to practice CSS Grid skills. This page is responsive in desktop, tablet, and mobile versions. The design for this project is from the Frontend Mentor website - here.
